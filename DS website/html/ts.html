Time and Space complexity

Time Complexity is the amount of time an algorithm takes to execute as a function of the input size. It can range from linear (O(n)) to exponential (O(2^n)). Space Complexity is the amount of memory an algorithm requires to execute as a function of the input size. It can range from constant (O(1)) to linear (O(n)) to exponential (O(2^n)).

Algorithm Time Complexity Space Complexity
Linear Search O(n) O(n)
Binary Search O(log n) O(1)
Merge Sort O(n log n) O(n)
Quicksort O(n log n) O(n)
Heap Sort O(n log n) O(n)
Hash Table O(1) O(n)
Tree (Binary Search Tree) O(h) O(n) or O(n log n)
Unsorted List O(n) O(1)

Time and space complexities are important in computer science because they help developers, analysts, and researchers to optimize their algorithms, reduce processing time, and use resources more efficiently.
